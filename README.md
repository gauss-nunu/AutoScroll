# AutoScroll

人間らしい自然な動作で自動スクロールを行うためのAndroidアプリケーションです。

バックグラウンドで動作するユーザー補助サービスとして、他のアプリ（TikTok, YouTube Shorts, etc.）の上にフローティングボタンを表示し、ボタンをタップするだけで自動スクロールのON/OFFを切り替えることができます。

## ✨ 主な特徴

このアプリは、自動化を検知されるリスクを最小限に抑えるため、「人間らしさ」の再現に特化しています。

- **カーブを描くスワイプ:** 直線的な機械の動きではなく、人間の指の動きに近い滑らかなベジェ曲線でスワイプします。
- **ランダムな軌道:** スワイプの始点・終点・カーブの軌道は、毎回少しずつランダムに変化します。
- **人間らしい「ゆらぎ」:** スクロール間隔とスワイプ速度には、常にランダムな「ゆらぎ」が加えられます。
- **柔軟な設定:** スクロール間隔、ゆらぎの大きさ、スワイプの速さなどを好みに合わせて細かく調整できます。

## 🔧 使い方

1.  **権限の許可:**
    - アプリを起動し、「Enable Accessibility Service」ボタンをタップして、設定画面から「AutoScroll」サービスを有効にします。
    - 「Enable Overlay」ボタンをタップし、「他のアプリの上に表示」を許可します。
2.  **フローティングボタンの起動:**
    - 権限を許可すると「Launch Floating Button」ボタンが有効になるので、これをタップします。
    - 画面上に緑色の再生ボタンが表示されます。このボタンはドラッグして好きな位置に移動できます。
3.  **自動スクロールの開始:**
    - スクロールさせたいアプリ（例: TikTok Lite）を開きます。
    - 緑色のフローティングボタンをタップすると、ボタンが赤色の停止ボタンに変わり、自動スクロールが開始します。
4.  **自動スクロールの停止:**
    - 赤色のフローティングボタンをタップすると、ボタンが緑色に戻り、自動スクロールが停止します。

## ⚙️ 設定項目

メイン画面のスライダーで、スクロールの挙動を細かく調整できます。

- **Base Delay (ベースディレイ):** 1回のスワイプが終わってから次のスワイプが始まるまでの基本的な待ち時間です (ms)。**この値を大きくすると、動画を長く視聴するようになります。**
- **Jitter Multiplier (ジッター係数):** スクロール間隔の「ゆらぎ」の大きさです。`0`だと一定間隔、`1`に近づけるほど間隔がランダムになります。
- **Swipe Length Variance (スワイプ長のばらつき):** スワイプする長さのランダム度合いです。
- **Swipe Duration (スワイプ時間):** 1回のスワイプ操作そのものにかける時間（速さ）です。
